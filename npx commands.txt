Cypress
========

1. Run all specific files in the project:

In the command line: 
	npx cypress run

Add the in package.json under scripts section add "test": "cypress run"  (for scripts we should use npm and not npx)
then 	
	npm test
	npm run test

2. Run all spec files in a folder
	npx cypress run --spec "cypress/integration/examples/**/*"

	npm run test --spec "cypress/integration/examples/**/*"


3. Run one spec file

npx cypress run --spec "cypress/integration/examples/actions.spec.js"

npm run test --spec "cypress/integration/examples/actions.spec.js"
